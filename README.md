

# Philosophy

> Simple web application to play the game: All roads lead to philosophy

## Requirement

- Python 3.7.1
- numpy 1.18.1
- pandas 0.25.3
- matplotlib 3.1.1
- sklearn 0.22.1
- Flask 1.1.1


## Usage example

run load_app.py:

    $ python  philosophie.py
    
open web browser to:  <a href="http://127.0.0.1:5000/" target="_blank">`http://127.0.0.1:5000/`</a>
